Detailed Description

Conscience's Body Collection.

Implementation: Each NSMutableDictionary holds the CoreGraphics drawing instructions to draw each state of each dynamic feature of the Conscience.

Object to be populated with the ConscienceLayers. Each Dictionary contains every expression frame for currently selected feature type. Member of ConscienceView.
