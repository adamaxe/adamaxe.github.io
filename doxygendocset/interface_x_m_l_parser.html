Detailed Description

XML Document Parser.

Implementation: We are parsing the svg's of the animatable facial features.

Common object to gather and transform Path Data from SVG. Limitations: SVG must be unoptimized, only small subset of SVG instructions are supported, only 1 level of grouping is supported currently, Points must be absolute
